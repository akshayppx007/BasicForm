# Basic form

## Getting Started

## create a .env file in server directory
DB_URI = YOUR_MONGODB_URL,  
NODE_ENV = PRODUCTION


### Client
To start the client application, follow these steps:  
   cd client,  
   npm install,  
   npm run dev

### Server   
To start the server application, follow these steps:  
   cd server,  
   npm install,  
   npm run dev ,
